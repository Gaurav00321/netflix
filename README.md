# Netflix UI Clone

A visually accurate clone of the Netflix user interface, created using only HTML and CSS. This project showcases modern front-end design techniques, responsive layout, and attention to detail.

## Features

- **Responsive Design:** Fully responsive layout compatible with various devices and screen sizes.
- **Custom UI Elements:** Styled navigation bar, hero banner, and content grids mimicking the Netflix interface.
- **Hover Effects:** Interactive hover animations for an engaging user experience.
- **Clean Code:** Organized HTML structure and reusable CSS classes for maintainability.

## Technologies Used

- **HTML5:** Semantic markup for a structured and accessible layout.
- **CSS3:** Modern styling with flexbox, grid layout, and media queries.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/netflix-ui-clone.git

2. Navigate to the project directory:
   ```bash
   cd netflix-ui-clone

3. Open the index.html file in your browser.

## Usage

- Explore the Netflix UI clone and experience its responsive design.
- Hover over the content grids and navigation items to view interactive effects.
